
# Health Prediction Application

## Tech Stack
- FastAPI
- Streamlit
- SQLite
- SQLAlchemy ORM
- OpenAI API

## Features
- CRUD Operations
- Input Validation
- Persistent Storage
- AI-generated Health Remarks
- Professional Project Structure
